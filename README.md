# Card Balance Dashboard

A privacy-focused dashboard for managing gift card codes and checking balances.

## Features
- **Secure Dashboard**: Password-protected access to stored cards.
- **Biometric Support**: Enable Face ID/Touch ID for easy login.
- **Privacy Mode**: Balance data is stored as $0.00 to prevent exposure while allowing card management.
- **Duplicate Prevention**: Intelligently blocks duplicate card submissions from the same device.
- **Real-time Updates**: Dashboard reflects changes instantly using Supabase Realtime.
- **Clean UI**: Modern, theme-aware design with optimized desktop and mobile views.

## Technologies
- Next.js
- Supabase
- Vanilla CSS
- Biometric Web Auth API
